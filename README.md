# RoadSafety — Безопасная дорога детства

Мобильное приложение для детей и родителей на Kotlin, предназначенное для обеспечения безопасности школьников при передвижении по городу.

## Описание проекта

Персональный навигатор безопасности для школьников и система мониторинга для родителей. Приложение использует систему цветовых зон (зеленая, желтая, красная) для оценки безопасности маршрута и поведения ребенка.

### Основные возможности (Must Have):
* **Система ролей:** Родитель или Ребенок (выбирается при регистрации).
* **Связывание аккаунтов:** Привязка/отвязка детей к родителям через OTP или QR-код.
* **Управление маршрутами:** Создание и редактирование безопасных маршрутов.
* **Контроль зон:** Родители могут размечать карту на зоны риска (красные, желтые, зеленые).
* **Мониторинг:** Отслеживание местоположения ребенка и уведомление о входе в опасную зону.
* **Система уведомлений:** Оповещения для родителей и критические алерты для детей.
* **Аварийная кнопка:** Быстрая запись видео/аудио ("кружок") и отправка родителям.
* **Чат:** Встроенный мессенджер между членами семьи.
* **Геймификация:** Начисление баллов и достижений за соблюдение правил безопасности.

## Технологический стек

* **Язык:** [Kotlin](https://kotlinlang.org/)
* **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Архитектура:** MVVM (Model-View-ViewModel)
* **DI:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
* **Сеть:** [Retrofit](https://square.github.io/retrofit/) + [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization)
* **Карты:** [MapLibre](https://maplibre.org/)
* **Локальное кэширование:** (Планируется Room/DataStore)

## Инструкция по запуску

Для сборки и запуска приложения вам понадобятся:
* **Android Studio** (рекомендуется версия Ladybug 2024.2.1 или новее).
* **JDK 17** (убедитесь, что он установлен и выбран в настройках проекта).
* Доступ к интернету для загрузки зависимостей и карт.

### Шаги для запуска:

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com/your-username/RoadSafetyApp.git
   cd RoadSafetyApp
   ```

2. **Открытие проекта:**
   * Запустите Android Studio.
   * Выберите **Open** и укажите путь к папке `RoadSafetyApp`.

3. **Синхронизация Gradle:**
   * Дождитесь завершения процесса **Gradle Sync**. Если возникнут ошибки, убедитесь, что у вас установлена правильная версия JDK в `Settings > Build, Execution, Deployment > Build Tools > Gradle`.

4. **Настройка эмулятора или устройства:**
   * Подключите физическое Android-устройство (с включенной отладкой по USB) или создайте виртуальное устройство (AVD) в **Device Manager**.
   * Рекомендуемая версия Android: 8.0 (API 26) и выше.

5. **Запуск:**
   * Выберите модуль `app` в конфигурации запуска.
   * Нажмите кнопку **Run** (зеленый треугольник).


## Структура проекта
* `app/src/main/java/team/kid/roadsafety/domain` — доменная логика и сущности.
* `app/src/main/java/team/kid/roadsafety/presentation` — UI слои на Compose.
* `app/src/main/java/team/kid/roadsafety/data` — репозитории и источники данных (API).
